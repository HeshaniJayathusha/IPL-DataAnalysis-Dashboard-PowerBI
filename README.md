IPL Data Analysis Dashboard (2008 - 2025)

📌 Project Overview:

This project provides a comprehensive visual analysis of the Indian Premier League (IPL) from its inception in 2008 up to the 2025 season. Using Power BI, I have transformed raw cricket data into an interactive dashboard that allows users to explore tournament trends, individual player performances, and team statistics.


📊 Key Features & Insights:

The dashboard provides a deep dive into various metrics, including:
Season Summary: Quick view of the Season Winner, Runner-up, and total matches played.
Batting Performance: Analysis of Total Sixes, Fours, Half-Centuries, and Centuries across seasons.
Individual Accolades: Tracking the Orange Cap (Leading Run-scorer) and Purple Cap (Leading Wicket-taker) holders.
Player Milestones: Identification of top players with the most 4s and 6s in a specific season.
Points Table: A detailed standings table showing Matches Played (Pld), Won, Lost, Net Run Rate (NRR), and Total Points.
